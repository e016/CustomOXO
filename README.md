# CustomOXO

![CustomOXO banner
](./docs/banner.png)

CustomOXO is a Tic-Tac-Toe (or Noughts and Crosses) game playable in your web browser. However, this version includes lots of customization options, like board shapes, player count, turn mechanisms and an optional "points" system for larger boards.
